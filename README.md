# Flight Management System

A Java desktop application for managing flight-related data, developed as a university project.

## Overview

This project is a Java application designed to manage and organize flight system data through a structured desktop interface.  
It includes separate modules for application logic, GUI, data storage, validation, and CSV-based data handling.

The goal of the project was to practice object-oriented design, desktop application structure, data validation, and modular project organization.

## Tech Stack

- Java
- Java Swing / desktop GUI
- Eclipse project structure
- CSV file handling
- Object-oriented programming

## Project Structure

- `src/app` – application logic and main flow
- `src/gui` – graphical user interface
- `src/model` – domain models and core entities
- `src/datastorage` – data storage logic
- `src/validation` – input and business rule validation
- `src/csv` – CSV import/export or parsing logic

## Features

- structured Java desktop application
- modular code organization
- GUI-based interaction
- data validation
- CSV-based data handling
- separation of logic, model, and interface layers

## Purpose

This project was created as part of coursework to practice Java programming, application architecture, and clean code organization in a desktop application.

## How to Run

1. Open the project in Eclipse.
2. Build the project.
3. Run the main application class from the `src/app` package.
4. Use the GUI to interact with the system.
